# Code of Conduct

## Our Pledge

We want this project to be a respectful, constructive place for collaboration.

Participants are expected to:

- be respectful and direct
- focus on technical substance
- avoid harassment, abuse, and personal attacks
- accept good-faith feedback

## Expected Behavior

- Use clear and professional language.
- Be specific about bugs, behavior, and tradeoffs.
- Respect privacy when discussing account data, sessions, and exports.
- Prefer concrete reproduction steps over vague complaints.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination
- Doxxing or sharing private account/session data
- Personal insults or hostile escalation
- Spam or deliberately misleading reports

## Enforcement

Project maintainers may remove comments, close threads, or block participation if behavior is harmful to the project or its contributors.

## Scope

This code of conduct applies to issues, pull requests, discussions, release comments, and other project communication channels.
